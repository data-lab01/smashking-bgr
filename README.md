A modern, responsive restaurant website for Smash King Burger – a handcrafted smashed burger joint in Nürnberg, Germany.

## Live

> **Live URL:** [https://data-lab01.github.io/smashking-bgr/](https://data-lab01.github.io/smashking-bgr/)

## Features

### Multi-Language Support
- **English (EN)** – Default language
- **German (DE)** – Localized for German customers
- **Arabic (AR)** – RTL support for Arabic speakers

### Menu System
- **Category filters** – Filter by Burgers, Sides, Drinks, Extras
- **6 Burger options** – Smokey King, Truffle King, Epic King, Golden King, Chicken King, Veggie options
- **Side dishes** – Fries, Nuggets, Onion Rings, Sweet Potato Fries, Chilli Cheese Nuggets
- **Beverages** – Coke, Sprite, Fanta, Lipton, Zero options, Mezzo Mix
- **Extras** – Extra patty, cheese, bacon, jalapeños, caramelized onions

### Shopping Cart
- Add/remove items from cart
- Quantity management
- Real-time price calculation
- Local storage persistence

### Build Your Own Burger
- Choose bun type (Brioche, Whole Wheat, Gluten Free)
- Choose patty (Beef, Chicken, Vegan, Double Beef)
- Add unlimited toppings with real-time price update

### Additional Pages
- **Locations** – Store addresses, hours, contact info
- **Deals & Offers** – Combo meals and promotions
- **About Us** – Brand story and statistics
- **Contact** – Contact form with validation

### Interactive Features
- **Store Status** – Real-time open/closed indicator based on business hours
- **Cookie Consent** – GDPR-compliant cookie banner with preferences
- **WhatsApp Order** – Floating button for quick WhatsApp orders
- **Mobile Menu** – Responsive hamburger menu for mobile devices
- **Image Gallery** – Interactive about page image gallery

### Responsive Design
- Fully responsive layout
- Works on desktop, tablet, and mobile devices
- Touch-friendly navigation
- RTL support for Arabic language
